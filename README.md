# Java-Program-Tarea-07-SENATI
En este repositorio quiero que conozcan mi trabajo en esta tarea sobre "Java Program".

## Java-Program-Tarea-07-SENATI
Sistema de registro de ventas para pequeñas empresas.
## 🛠️ Requisitos Previos
Para ejecutar este proyecto de forma óptima, es necesario contar con el siguiente entorno:

* Apache NetBeans IDE: Versión 12.0 o superior recomendada.
* Java Development Kit (JDK): Versión 8 o superior.
* Git: Opcional, para clonar el repositorio. [4, 5, 6] 

## 🚀 Instalación y Ejecución
Sigue estos pasos para abrir el proyecto en NetBeans:

   1. Descargar el proyecto: Clona el repositorio o descarga la carpeta con los archivos fuente. [6, 7] 
   2. Abrir NetBeans: Inicia el IDE. [2, 8, 9] 
   3. Importar el proyecto:
   * Ve al menú superior: File > Open Project... (o presiona Ctrl + Shift + O).
      * Navega hasta la carpeta descargada. NetBeans reconocerá automáticamente el icono del proyecto (un icono de taza de café sobre una carpeta) si detecta los archivos de configuración (nbproject, pom.xml o build.xml).
      * Haz clic en Open Project. [6, 10] 
   4. Ejecutar:
   * Una vez cargado, haz clic derecho sobre el nombre del proyecto en el panel izquierdo y selecciona Run o pulsa la tecla F6. [11]

------------------------------

Nota: Este proyecto fue desarrollado bajo el esquema de construcción Ant (o Maven/Gradle), asegúrate de que tu instalación de NetBeans incluya soporte para este tipo de proyectos. [10, 13]
